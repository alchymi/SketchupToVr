VR Export Pipeline

SketchUp → GLB → VR Loader

Cette plateforme permet l’envoi et la gestion de fichiers .glb exportés depuis SketchUp via le plugin LMDDC VR Exporter.

Les fichiers peuvent ensuite être visualisé via l'app Sketchup to VR Unity pour charger dynamiquement des environnements 3D.
